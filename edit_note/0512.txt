add : Menu Functions를 추가하여 메뉴 화면을 구성하였습니다.
- draw_logo 함수는 로고를 화면 중앙에 출력하며, 로고의 가로 및 세로 크기를 계산하여 적절한 위치에 배치합니다.
- get_logo_w_size 함수는 로고의 가장 긴 문자열의 길이를 계산하여 반환합니다.
- SetColor 함수는 ncurses의 색상 쌍을 초기화하여 메뉴와 게임 화면에서 사용할 다양한 색상을 설정합니다.

add : Main Function을 작성하여 프로그램의 메인 루프를 구성하였습니다.
- STATE_MENU 상태에서는 메뉴 화면을 출력하며, 사용자가 화살표 키로 메뉴를 탐색하고 Enter 키를 눌러 선택할 수 있습니다.
- STATE_INFO 상태에서는 게임의 간단한 설명과 개발자 정보를 출력합니다. q 키를 눌러 메뉴로 돌아갈 수 있습니다.
- STATE_GAME 상태는 게임 화면을 처리하며, 현재는 기본적인 박스만 출력됩니다.
- STATE_EXIT 상태에서는 ncurses 모드를 종료하고 프로그램을 종료합니다.

add : Maze Functions를 추가하여 미로를 생성하고 출력하는 기능을 구현하였습니다.
- generate_maze 함수는 BFS(너비 우선 탐색)를 사용하여 미로를 생성하며, 모든 셀을 벽으로 초기화한 후 경로를 생성합니다.
- print_maze 함수는 생성된 미로를 화면에 출력하며, 벽은 #, 경로는 공백으로 표시됩니다.

add : Utility Functions를 추가하여 프로그램에서 자주 사용되는 기능을 구현하였습니다.
- shuffle 함수는 배열의 요소를 랜덤하게 섞으며, 미로 생성 시 방향을 랜덤하게 탐색하기 위해 사용됩니다.
- str_len 함수는 문자열의 길이를 계산하여 반환하며, 로고 출력 시 문자열의 길이를 계산하는 데 사용됩니다.

add : 전역 변수 EXIT와 key_pressed를 추가하여 프로그램 종료 여부와 사용자 입력을 관리합니다.
- EXIT는 프로그램 종료 여부를 나타내며, true로 설정되면 메인 루프가 종료됩니다.
- key_pressed는 사용자가 입력한 키를 저장하며, 메뉴 탐색 및 선택 시 사용됩니다.

add : game_states 열거형을 추가하여 프로그램의 상태를 관리합니다.
- STATE_MENU, STATE_INFO, STATE_GAME, STATE_EXIT의 네 가지 상태를 정의하였으며, 상태 전환은 key_pressed와 메뉴 항목 선택에 따라 이루어집니다.
